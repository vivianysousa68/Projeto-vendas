# 🍰 Delícias da Vih

Sistema de vendas online de doces com carrinho de compras e finalização de pedido via WhatsApp.

## 📌 Sobre o projeto
O "Delícias da Vih" é um sistema desenvolvido para facilitar a venda de produtos doces de forma prática e rápida. O usuário pode visualizar os produtos, adicioná-los ao carrinho e finalizar o pedido diretamente pelo WhatsApp.

## 🎯 Objetivo
Automatizar o processo de pedidos e melhorar a experiência do cliente, oferecendo uma forma simples de compra online.

## ⚙️ Funcionalidades
- Visualização de produtos
- Adição e remoção de itens no carrinho
- Cálculo automático do total
- Integração com WhatsApp para envio do pedido

## 💻 Tecnologias utilizadas
- HTML
- CSS
- PHP

## 📱 Como funciona
O cliente escolhe os produtos, adiciona ao carrinho e, ao finalizar, é redirecionado para o WhatsApp com uma mensagem pronta contendo o pedido.

## 👥 Público-alvo
Pessoas interessadas em comprar doces de forma rápida e prática pela internet.
